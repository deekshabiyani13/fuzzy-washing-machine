# Fuzzy Logic Washing Machine System

##  Description
This project implements a fuzzy logic system using MATLAB to determine washing time based on dirt level and load size.

##  Inputs
- Dirt Level (0–10)
- Load Size (0–10)

##  Output
- Wash Time (Short, Medium, Long)

##  Methodology
- Mamdani Fuzzy Inference System
- Triangular and Trapezoidal Membership Functions
- 9 Fuzzy Rules

##  Results
- Smooth variation in output
- Surface viewer shows nonlinear relationship
- Rule viewer demonstrates inference

##  Files Included
- MATLAB code (.m)
- FIS file (.fis)
- Output screenshots

##  Note
This system is implemented programmatically using MATLAB, demonstrating deeper understanding beyond GUI-based design.
